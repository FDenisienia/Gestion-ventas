# Sistema de Gestión de Clientes y Ventas

Aplicación web para gestionar ventas y clientes construida con React, Vite y Node.js.

## Estructura del Proyecto

- `frontend/` - Aplicación React con Vite
- `backend/` - API REST con Node.js y Express

## Instalación

1. Instalar dependencias de todos los proyectos:
```bash
npm run install:all
```

## Desarrollo

Para ejecutar tanto el frontend como el backend simultáneamente:
```bash
npm run dev
```

O por separado:
- Frontend: `npm run dev:frontend` (puerto 3000)
- Backend: `npm run dev:backend` (puerto 3001)

## Características

- Header vertical a la izquierda con navegación
- Rutas configuradas para todas las secciones
- Diseño moderno y responsivo
